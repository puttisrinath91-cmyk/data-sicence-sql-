# 📚 Library Management System

A modern **Library Management System** built using **PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap**. The system simplifies library operations by providing separate dashboards for administrators and students, allowing efficient management of books, users, borrowing records, and document uploads.

---

## 🚀 Features

### 👨‍💼 Admin Module

* Admin Login
* Dashboard Overview
* Add, Update & Delete Books
* Manage Students
* Issue & Return Books
* View Borrowing History
* Upload Books & Documents
* Search Functionality

### 👨‍🎓 Student Module

* Student Login
* Dashboard
* Search Books
* Borrow Books
* Return Books
* View Borrowing History
* View Uploaded Documents
* Profile Management

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript, Bootstrap
* **Backend:** PHP
* **Database:** MySQL
* **Server:** XAMPP (Apache & MySQL)

## ⚙️ Installation

### 1. Clone the Repository

```bash
git clone https://github.com/gnaneshwar8143/Library-Management-System.git
```

### 2. Move the Project

Copy the project folder to:

```
xampp/htdocs/
```

### 3. Import Database

* Open **phpMyAdmin**
* Create a new database
* Import the provided SQL file

### 4. Start XAMPP

Start:

* Apache
* MySQL

### 5. Run the Project

Open your browser:

```
http://localhost/Library_Management/
```
